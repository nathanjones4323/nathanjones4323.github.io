I'm a Senior Data Engineer based in Ventura, California, with a background in Financial Mathematics & Statistics from UC Santa Barbara. Since 2020 I've been at Curri, a construction materials delivery platform, where I build the data infrastructure that powers business decisions at scale.

My toolkit spans the full data stack — PostgreSQL pipelines, dbt models, analytics engineering — and extends into full-stack development with Python, FastAPI, and React Native. Featured AI-Powered Full-Stack

AI Meal Planner

A full-stack nutrition app where users chat with an AI agent to generate personalized meal plans. Claude autonomously searches the USDA FoodData Central database, calculates precise nutrient profiles, and assembles week-long plans complete with recipes and step-by-step cooking instructions — all streamed live to the UI as the agent works. Built with a FastAPI backend, cross-platform React Native frontend, and a fully migrated PostgreSQL schema.

FastAPI React Native Expo Anthropic Claude API PostgreSQL USDA FDC API Google OAuth Docker TypeScript NativeWind Alembic GitHub Actions
View on GitHub
🤖
Agentic AI
Claude uses three custom tools — search foods, fetch nutrition data, and persist meal plans — orchestrating multi-step reasoning on every request.
📡
Real-Time Streaming
Server-sent events stream each agent step live to the chat UI, showing which tools fired and what data was fetched as it happens.
🗄️
Production Database
Seven-table PostgreSQL schema — users, profiles, conversations, foods, meal plans, meals, and meal foods — managed with full Alembic migration history.
📱
Cross-Platform Mobile
Single React Native codebase targets iOS, Android, and Web via Expo Router with NativeWind styling and shimmer loading animations.
🔐
Auth & Secrets
Google OAuth handles user authentication while SOPS and age encryption secure all secrets in the CI/CD pipeline — no credentials in plaintext.
🚀
Production-Ready DevOps
Fully containerized with Docker Compose, wired to GitHub Actions for CI, and using psycopg connection pooling for high-throughput database access.
